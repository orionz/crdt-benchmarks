
# setup

## Install automerge-backend-wasm locally 

(there is no package)

```
  $ git clone git@github.com:automerge/automerge-rs.git
  $ cd automerge-rs/automerge-backend-wasm
  $ yarn release
  $ yarn link
```

## Setup the benchmark

```
  $ npm install
  $ yarn link automerge-backend-wasm
  $ npm start
```
